1. What is the difference between map, filter, foreach, find?
Ans: map() executes the same code on every element in an array and returns a new array with the updated elements. filter() checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria. forEach(), is used to execute the same code on every element in an array but does not change the array and it returns undefined.find() is used to select a single element from an array.

2. What is the difference between local storage,session storage and cookies?
Ans: Local storage is the most recent mechanism. It allows for larger amounts of data to be stored, but the data is not deleted when the browser is closed. Local storage is useful for storing data that the user will need to access later, such as offline data.Session storage is similar to cookies, but the data is only stored for the current session. This means that the data will be deleted when the user closes the browser. Session storage is useful for storing data that is sensitive, such as login credentials.Cookies are the oldest and most well-known mechanism. They are simple to use and well supported by browsers.

3.What is Event loop and how it works?
Ans: The event loop is the secret behind JavaScript's asynchronous programming. The event loop is a process that waits for the Call Stack to be clear before pushing callbacks from the Task Queue to the Call Stack. Once the Stack is clear, the event loop triggers and checks the Task Queue for available callbacks. If there are any, it pushes it to the Call Stack, waits for the Call Stack to be clear again, and repeats the same process.

5. What is the work of function chooseQuiz
Ans: Here chooseQuiz function is called when the user selects an answer for a particular quiz question in the website. The function takes two parameters index and givenAns. The function checks if there is an existing answer for the quiz question. If an answer exists for the quiz question, the function updates the answer with the new answer chosen by the user. If an answer does not exist for the quiz question, the function adds a new object to the answers array containing the quiz question and the answer chosen by the user.
